# 🧑‍⚕️ Mahajan'c Clinic

A modern web application built with **React, TypeScript, Vite, Tailwind CSS**, and **shadcn/ui**.

## 📌 Project Overview

This project is a fast, scalable, and developer-friendly frontend application.  
It uses modern tooling and best practices to ensure performance, maintainability, and a great developer experience.

---

## 🛠️ Tech Stack

- **Vite** – Fast build tool
- **React** – UI library
- **TypeScript** – Type safety
- **Tailwind CSS** – Utility-first CSS framework
- **shadcn/ui** – Accessible & customizable UI components

---

## ⚙️ Getting Started

Follow the steps below to run the project locally.

### ✅ Prerequisites

Make sure you have the following installed:

- **Node.js** (v18+ recommended)
- **npm** or **yarn**
